CL Flunky


Automated Craigslist Data Scraping and Notification

David Shuman
NSS Cohort 12
April 5, 2016

Project Goals:

  • Pull data from website that provides no API
  • Schedule repeated searches at short intervals
  • Manage saved searches with CRUD application
  • Prompt notification of search hits via SMS

Principal Components

  • Firebase API
    • User authentication
    • Store saved search parameters
    • Securely store Twilio username and password
  • Twilio REST API accepts outgoing text messages via POST
  • Original code harvests data from last 100 posts to Craigslist "for sale" page
  • CRUD application to run, schedule, and manage multiple searches

Lessons Learned:

  • Maintain realistic expectations
  • Manage time; prioritize
  • Write pseudo-code!
  • Don't be afraid to junk code that is more easily rewritten than fixed
